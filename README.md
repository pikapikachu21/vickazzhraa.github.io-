## 🚀 Cara Upload ke GitHub Pages

### Langkah 1 — Buat Repository
1. Login ke [github.com](https://github.com)
2. Klik **New repository**
3. Nama repository: **`vickazzhraa.github.io`** (ganti dengan username GitHub kamu)
4. Set ke **Public**
5. Klik **Create repository**

### Langkah 2 — Upload Semua File
1. Di halaman repository, klik **Add file → Upload files**
2. Upload semua file dan folder (index.html, index-en.html, README.md, dan folder assets/)
3. Klik **Commit changes**

> ⚠️ **Penting:** Pastikan folder `assets/` beserta semua isinya ikut ter-upload!

### Langkah 3 — Aktifkan GitHub Pages
1. Klik tab **Settings**
2. Scroll ke bagian **Pages**
3. Di **Source**, pilih: `Deploy from a branch`
4. Branch: `main` → folder: `/ (root)`
5. Klik **Save**

### Langkah 4 — Website Live! 🎉
Tunggu 1–2 menit, lalu buka:
```
https://vickazzhraa.github.io
```

## 📝 Cara Edit Konten

### Ganti Teks
Buka `index.html` dengan text editor (Notepad, VSCode, dll), cari teks yang ingin diubah, edit, lalu save.

### Ganti Foto
Ganti file foto di folder `assets/images/` dengan foto baru menggunakan nama file yang **sama persis**.

### Ganti CV
Ganti file `assets/pdf/CV_Vicka_Azzahra.pdf` dengan versi CV terbaru.
